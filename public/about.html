About

Me#

Hi! My name is Reilly.

I’m originally from the foothills of northern California, and I studied physics (BSc) at UC Santa Barbara’s College of Creative Studies (my undergraduate thesis1 addressed errors in high-frequency, analog signal processing used for controlling a superconducting quantum computer).

I’ve spent the last few years travelling (Eastern Asia, Eastern Australia, and North America) while reading extensively, pursuing a few personal projects, and working in software.

This Site#

Non-Technical#

This is mostly a space for me share my fascination with various subjects, to publish personal essays and to communicate about projects or developments which are important to me.

Technical#

I’m using pelican to render this site from markdown and hosting the result via github pages.

My domain is registered with Namecheap, which points to cloudflare‘s nameservers, which have DNS A records pointing to the IP address for the github pages servers, where the site is hosted as a public git repository.

At the end of the day, I write in Markdown with a few custom extensions, use a script that builds and previews the site locally, (with python and node.js), and I publish my changes by pushing to the hosted git repo.

The relevant code is hosted publically on github for anyone to use, under the MIT license. It is not perfect, but seeks to avoid as much bloat as possible while doing what I want it to. Please, use and hack it to your heart’s content.
